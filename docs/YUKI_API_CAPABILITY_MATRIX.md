# Yuki API Capability Matrix

Source references:
- Main docs: https://documenter.getpostman.com/view/12207912/UVCBB51L
- Live ASMX/WSDL inspected on 2026-05-07.

Notes:
- Yuki webservices are SOAP/XML and ASMX-based, not REST-first.
- Common session methods are exposed on most services: Authenticate, Domains, GetCurrentDomain, SetCurrentDomain, AdministrationID, Administrations.
- Daily call limit guidance from Yuki docs: 1000 calls/domain/day baseline.
